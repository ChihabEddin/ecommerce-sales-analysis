📊 E-commerce Sales Analysis 
📌 Project Overview

This project analyzes an e-commerce sales dataset to extract meaningful business insights using Python and Exploratory Data Analysis (EDA) techniques.

The goal is to understand sales performance, customer behavior, and business trends to support better decision-making.

📂 Dataset Information
Dataset: E-commerce Sales Data (CSV format)
Records: ~5,000 transactions
Time Period: 2023 – 2024
Features include:
Order Date
Customer Segment
Product Category
Sales
Profit
Region
Discount
Ship Mode

🎯 Objectives
Analyze overall sales performance
Identify top-performing categories and sub-categories
Understand profit distribution
Study customer segments
Analyze regional performance
Evaluate impact of discounts

🛠️ Tools & Technologies
Python 🐍
Pandas
Matplotlib
Seaborn
Jupyter Notebook

📊 Exploratory Data Analysis (EDA)

The project includes the following analyses:

1️⃣ Sales Analysis
Monthly Sales Trends
Sales by Category
Sales by Sub-Category

2️⃣ Profit Analysis
Profit by Category
Profit by Sub-Category

3️⃣ Customer Analysis
Sales and Profit by Customer Segment
Top Customers by Sales

4️⃣ Business Insights
Regional Sales Performance
Discount vs Profit Analysis
Shipping Mode Performance

📈 Key Insights
Technology is the highest revenue-generating category
Certain customer segments generate higher sales but lower profit
Discounts negatively impact profitability
Some regions perform significantly better than others
Shipping methods affect delivery performance
